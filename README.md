# Focus Timer

A small interactive pomodoro-style timer with hidden puzzle elements, ambient sound toggles, and an eerie final sequence.

## Features

- 3-minute work timer and 1-minute break timer
- Start, pause, and reset controls
- Animated progress ring
- Ambient sound buttons for rain and fireplace
- Hidden secret puzzle with layered unlocks
- Unsettling visual changes as layers progress
- Final secret interaction that triggers a dramatic leave message

## Files

- index.html — main structure of the page
- style.css — styling, layout, and visual effects
- script.js — timer logic, sound controls, puzzle progression, and final sequence

## How to Run

Open index.html in your browser, or serve the folder with a simple local server if you prefer.

Example:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Notes

Some browser autoplay restrictions may affect ambient audio playback. If audio does not start immediately, interact with the page first and try the sound button again.
